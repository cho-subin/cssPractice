# cssPractice - React
당근마켓 모바일용 main view 클론코딩을 통한 css 연습

### 👀 BEM 명명법
BEM : Block-Element-Modifiter 세가지로 구성된 이름을 짓는다.
<br>
각각 __와 --로 구분한다.
<br>
ex) info-bar__div--small

### 👀 Style reset
크로스 브라우징을 위해서 사용한다.
<br>
npm 설치
<br>
`
$ npm i styled-reset
`
<br>
`
$ npm i --save-dev @types/styled-components
`
